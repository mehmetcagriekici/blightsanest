# Contributing

Contributions are welcome. Please fork the repository and open a pull request to the `main` branch.

For bugs, questions, or unclear documentation reach out at **mehmetcagriekici@gmail.com**

---

## Steps

**1. Fork the repository** on GitHub, then clone your fork:

```bash
git clone https://github.com/<your-username>/blightsanest.git
cd blightsanest
```

**2. Run the tests**

```bash
go test ./...
```

**3. Submit a pull request** from your fork to the `main` branch.
